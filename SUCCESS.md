- ✅ 绿色勾 - 构建成功
- ❌ 红色叉 - 构建失败

### 2. 等待构建完成（5-10分钟）

首次构建需要下载依赖，会比较慢。

您可以：
- 刷新Actions页面查看进度
- 点击构建记录查看详细日志
- 等待黄色圆点变成绿色勾

### 3. 下载APK

构建成功后：

1. 点击构建记录（最近的那个）
2. 滚动到页面底部
3. 找到 "Artifacts" 部分
4. 点击 `app-debug-apk`
5. 下载ZIP文件（约3-5MB）
6. 解压ZIP文件
7. 得到 `app-debug.apk`

### 4. 安装到手机

```bash
# 使用ADB安装
adb install app-debug.apk

# 或直接传输到手机，用文件管理器安装
```

### 5. 测试功能

安装后测试：
- ✅ 添加位置点
